# 新規座席予約
# 機能概要
新規座席予約を行う。  
1. 期間の指定
2. 空席確認
3. 予約する座席の選択
4. 予約するユーザーの選択
5. 予約確定

# 画面項目説明
* 座席表  
座席が空いているかどうかが表示される。  
予約したい席を選択する。
* 開始日  
予約開始日を指定する。  
日付ピッカーを使用する。  
デフォルト値は本日の日付。
* 終了日  
予約終了日を指定する。  
日付ピッカーを使用する。  
デフォルト値は本日の日付。
* 指定期間の空席状況確認　- ボタン  
開始日～終了日の間がすべて空いている席を表示するように、座席表を更新する。  
* 予約ユーザー  
予約したいユーザーを選択する。  
デフォルトでは、空。  
ユーザー一覧の小画面が表示され、そこで選択したユーザーの名前が入力される。
* キャンセル - ボタン  
座席表画面に戻る
* 予約確定 - ボタン  
予約を実行する。
# 機能要件
* 画面の初期表示時、本日の日付で空席状況を表示しておく。
* 空席状況確認ボタン押下時、以下のチェックを行い、該当するエラーメッセージを出力する。  
エラーがない場合は、座席表の更新を行う。  
    * 開始日、終了日は必須。  
    メッセージID='E0002' 、\<item1\> ＝ 項目名 
    * 指定期間が逆転している場合  
    メッセージID='E0009'
    * 不正な日付を入力  
    メッセージID='E0008'  
* 予約済の座席は非活性となり、選択できない。  
* 選択できる座席は１つのみ。  
１つ選択した状態で別の座席を選択した場合は、もともと選択していた座席の選択は解除される。
* 選択した状態で、空席状況確認ボタンを押下した場合、選択はリセットされる。
* 予約ユーザー欄を選択したとき、ユーザー一覧画面のダイアログが表示される。  
そこで選択したユーザーが設定される。
* 予約確定ボタン押下時、以下のチェックを行い、該当するエラーメッセージを出力する。  
エラーがない場合は、座席予約を行う。  
    * 開始日、終了日、座席、予約ユーザーは必須。  
    メッセージID='E0002' 、\<item1\> ＝ 項目名 
    * 指定期間が逆転している場合  
    メッセージID='E0009'
    * 不正な日付を入力  
    メッセージID='E0008'  
    * 予約が重複した場合(サーバー側でのみチェック)  
    メッセージID='E0010'
